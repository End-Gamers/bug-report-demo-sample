프로바이더별 설정 안내
Groq (무료, 신용카드 불필요)
console.groq.com에서 API 키(gsk_...)를 발급합니다. 분당 30회, 하루 1,000회 요청 가능.
Anthropic Claude
console.anthropic.com에서 API 키(sk-ant-...)를 발급합니다.
AWS Bedrock API Key
AWS 콘솔 → Amazon Bedrock → API keys에서 발급합니다.
장기 키: ABSK... / 단기 키: bedrock-api-key-...
AWS Bedrock IAM
IAM에서 bedrock:InvokeModel 권한이 있는 사용자의 Access Key를 생성합니다. 리전에서 Claude 3.5 Sonnet v2 모델 접근이 활성화되어 있어야 합니다.
